# How to build it? 

	- Need to install lastest version Node.Js : https://nodejs.org/en/

	- Install mongodb: https://docs.mongodb.com/v3.2/installation/

	- Run by commandline:

		* open terminal and cd to folder of project 

		* $run: npm install

		* $run: gulp
